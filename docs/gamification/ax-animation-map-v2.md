# AX Animation Map (v2)

## Motion style
- Fast, sharp, modern.
- Micro animations: 180-260ms.
- Milestone animations: 320-520ms.

## Event map
1. Question correct
- Effect: subtle card pulse + `+AX` float.
- Duration: 220ms.

2. Question incorrect
- Effect: low-amplitude shake + retry hint fade.
- Duration: 240ms.

3. Route completed
- Effect: progress ring sweep + AX burst.
- Duration: 420ms.

4. Skill -> Developing
- Effect: status card glow + rank nudge.
- Duration: 420ms.

5. Skill -> Secure
- Effect: controlled confetti + AX spin-in.
- Duration: 500ms.

6. Streak extended
- Effect: streak icon ignite + count tick.
- Duration: 260ms.

## Accessibility
- Reduced motion mode: fade-only alternatives.
- Disable particles in low-performance mode.
