# AX Reward Economy (v2)

## Base rewards
| Event | AX |
|---|---:|
| Diagnostic question correct | +5 |
| Diagnostic question attempted (incorrect) | +2 |
| Shadow question correct | +8 |
| Route completed | +20 |
| Skill -> Developing | +30 |
| Skill -> Secure | +60 |
| Recovery bonus (wrong then right) | +25 |
| Daily streak maintained | +15 |
| Weekly target hit | +80 |

## Multipliers
- Session accuracy >=85%: +10% AX session bonus.
- 2+ secure transitions in one session: +1 token-style bonus equivalent (optional).

## Anti-spam safeguard
If 3 rapid wrong submissions (<6s each):
- halve AX for next 5 questions.

## Spend categories (cosmetic)
- Theme skin: 60 AX
- Avatar frame: 80 AX
- Progress trail: 100 AX
- Title badge tier: 120 AX

## Integrity rules
- No AX for abandoned/invalid submissions.
- No negative AX deductions for wrong answers.
- Log all grants with reason + event id.
